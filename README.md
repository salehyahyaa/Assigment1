# Fermat Near Miss Search

A Python program that searches for near misses of Fermat’s Last Theorem:

x^n + y^n ≠ z^n

The program finds values of x, y, and z that come closest to satisfying the equation for a given n and range limit k.

## Features
- Takes user input for n and k  
- Tests all x and y values in range  
- Computes closest z and corresponding miss  
- Tracks and prints the smallest relative miss  

## How to Run
1. Make sure Python 3 is installed  
2. Run:
   python3 main.py
3. Enter n and k when prompted  

## Requirements
- Python 3
